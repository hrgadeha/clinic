# clinic

Steps To start using Clinic App-

1. Install ERPNext with "Distribution" domain

2. Add Consultant item in clinic setting(Healthcare Setting)

3. Add Consultation charge and cost center in doctor master which is used duing making invoice.

4. You should add appointment field in sales invoice because that is available only in healthcare module so if you install app in            different module then you should add this column.

5. custom translation file available at /clinic/clinic/transalation.csv
